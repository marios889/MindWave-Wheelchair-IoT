# Smart Home Interface

This is a Flutter application for controlling smart home devices.

## Prerequisites

Before running the app, ensure you have the following installed:

- [Flutter SDK](https://flutter.dev/docs/get-started/install)
- A code editor like [Visual Studio Code](https://code.visualstudio.com/) with Flutter extensions
- Android Studio for Android development

## Installation

1. Clone the repository or navigate to the project directory:
   ```
   cd smart_home_interface
   ```

2. Install dependencies:
   ```
   flutter pub get
   ```

## Running the App

### For Android

1. Connect an Android device or start an Android emulator.
2. Run the app:
   ```
   flutter run
   ```

## Building the App

To build a release APK for Android:

```
flutter build apk --release
```

## Additional Notes

- Ensure your Flutter version is up to date: `flutter upgrade`
- Check for any Android-specific setup in the Flutter documentation.
